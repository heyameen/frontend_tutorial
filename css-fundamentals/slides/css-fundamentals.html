CSS Fundamentals

Styling the Web

What is CSS?

  • CSS = Cascading Style Sheets
  • Purpose: Style and layout web pages
  • Analogy: HTML = structure, CSS = design

How CSS Works

  • Selects HTML elements
  • Applies styles to selected elements
  • Cascading: Multiple styles can apply to one element

Linking CSS to HTML

  1. Inline CSS 
    <p style="color: blue;">Blue text</p>
  2. Internal CSS 
    <style>
  p { color: blue; }
</style>
  3. External CSS (Recommended) 
    <link rel="stylesheet" href="styles.css">

CSS Selectors

  • The parts of a CSS rule
  • Simple selectors: .classname { }
  • Pseudo-classes and pseudo-elements: #idname:hover { }
  • Complex selectors: .top * + * { }

Common CSS Properties

  • Color: color, background-color
  • Font: font-family, font-size, font-weight
  • Text: text-align, text-decoration
  • Box: width, height, margin, padding
