HTML document structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Explanation of each part:

  • <!DOCTYPE html>: Declares that this is an HTML5 document
  • <html>: The root element of the HTML page
  • <head>: Contains meta-information about the document
  • <meta charset="UTF-8">: Specifies the character encoding
  • <meta name="viewport">: Ensures proper rendering on mobile devices
  • <title>: Specifies the title of the document (shown in browser tab)
  • <body>: Contains the visible content of the document

Common HTML elements

Headings:

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>
<!-- ... up to h6 -->

Paragraphs:

<p>This is a paragraph of text.</p>

Links:

<a href="https://www.example.com">Visit Example.com</a>

Images:

<img src="image.jpg" alt="Description of the image">

Lists:

<ul>
    <li>Unordered list item</li>
    <li>Another item</li>
</ul>

<ol>
    <li>Ordered list item</li>
    <li>Another item</li>
</ol>

Tables:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

Forms:

<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <button type="submit">Submit</button>
</form>

Semantic HTML

Semantic elements provide meaning to the structure of web content:

  • <header>: Introductory content or navigation links
  • <nav>: Navigation links
  • <main>: Main content of the document
  • <article>: Self-contained composition (e.g., blog post, news article)
  • <section>: Standalone section of content
  • <aside>: Content tangentially related to the main content
  • <footer>: Footer of a document or section

Example of semantic HTML structure:

<body>
    <header>
        <h1>Website Title</h1>
        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
                <li><a href="#contact">Contact</a></li>
            </ul>
        </nav>
    </header> 
     
    <main>
        <article>
            <h2>Article Title</h2>
            <p>Article content...</p>
        </article>
        
        <section>
            <h2>Section Title</h2>
            <p>Section content...</p>
        </section>
    </main>
    
    <aside>
        <h3>Related Links</h3>
        <ul>
            <li><a href="#">Link 1</a></li>
            <li><a href="#">Link 2</a></li>
        </ul>
    </aside>
    
    <footer>
        <p>&copy; 2024 Your Company. All rights reserved.</p>
    </footer>
</body>

Using semantic HTML improves accessibility, search engine optimization (SEO), and code readability. It provides a clear structure to your document, making it easier for both humans and machines to understand the content and its organization.
