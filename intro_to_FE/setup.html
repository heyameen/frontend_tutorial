Installing a code editor (VSCode recommended)

  1. Download VSCode from the official website (https://code.visualstudio.com/)
  2. Run the installer and follow the prompts
  3. Launch VSCode and familiarize yourself with the interface
  4. Install useful extensions:
    • Live Server (for local development server)
    • Prettier (code formatter)
    • ESLint (JavaScript linter)
    • Auto Rename Tag (for HTML editing)
    • Color Highlight (for visualizing colors in code)

Browser setup and extensions

  1. Install Google Chrome or Mozilla Firefox
  2. Set up developer tools:
    • Chrome: Menu > More Tools > Developer Tools (or F12)
    • Firefox: Menu > Web Developer > Toggle Tools (or F12)
  3. Install useful browser extensions:
    • Web Developer: Adds a toolbar with various web development tools
    • ColorZilla: Color picker and eyedropper tool
    • Responsive Web Design Tester: Test websites at different screen sizes
    • JSON Formatter: Makes JSON data more readable
    • Lighthouse: Audits for performance, accessibility, and best practices

Introduction to version control (Git) and creating a GitHub account

  1. Install Git from https://git-scm.com/
  2. Set up Git with your name and email: 
    git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
  3. Create a GitHub account at https://github.com/
  4. Basic Git workflow:
    • Initialize a repository: git init
    • Add files to staging: git add .
    • Commit changes: git commit -m "Commit message"
    • Create a remote repository on GitHub
    • Link local repository to remote: git remote add origin <repository-url>
    • Push to remote repository: git push -u origin main
  5. Collaborative Git features:
    • Branching and merging
    • Pull requests
    • Code reviews
    • Issue tracking
