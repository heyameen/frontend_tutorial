Text editors

Visual Studio Code (VSCode):

  • Free, open-source code editor developed by Microsoft
  • Highly customizable with a vast ecosystem of extensions
  • Features:
    • IntelliSense for smart completions
    • Debugging support for various languages
    • Built-in Git integration
    • Integrated terminal
    • Live Share for collaborative coding
    • Customizable themes and keyboard shortcuts

Sublime Text:

  • Lightweight and fast text editor
  • Known for its speed and minimalist interface
  • Features:
    • Multiple selections for simultaneous editing
    • Powerful search and replace functionality
    • Customizable through packages and plugins
    • Distraction-free writing mode
    • Split editing for working on multiple files

Browsers and developer tools

Google Chrome:

  • WebKit-based browser with a large market share
  • Chrome DevTools offer powerful debugging and performance analysis tools
  • Features of Chrome DevTools:
    • Elements panel for inspecting and modifying the DOM and CSS
    • Console for JavaScript debugging and logging
    • Network panel for analyzing HTTP requests and responses
    • Performance and Memory tools for optimizing website speed
    • Application panel for working with storage, service workers, and PWAs
    • Mobile device simulation for responsive design testing

Mozilla Firefox:

  • Open-source browser with a focus on privacy and customization
  • Firefox Developer Tools provide similar functionality to Chrome DevTools
  • Unique features:
    • Responsive Design Mode for testing various screen sizes
    • Accessibility inspector for checking web accessibility
    • CSS Grid inspector for visualizing and debugging grid layouts

Command line basics

The command line interface (CLI) is a text-based way to interact with your computer. It's an essential tool for developers, offering more power and flexibility than graphical interfaces for many tasks.

Basic commands:

Navigation:

  • cd (change directory)
  • ls / dir (list contents)
  • pwd (print working directory)

File operations:

  • touch / echo (create files)
  • mkdir (create directories)
  • rm / del (remove files/directories)
  • cp (copy files)
  • mv (move/rename files)

Text manipulation:

  • cat / type (display file contents)
  • grep / findstr (search text)

Package management:

  • npm (Node Package Manager)
  • yarn (alternative to npm)

Version control:

  • git commands (clone, add, commit, push, pull)
